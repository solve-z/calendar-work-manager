Privacy Policy

Calendar Work Manager is a Chrome / Edge browser extension that helps users create and manage work-related tasks directly in Google Calendar.

Last updated: December 22, 2025

1. Information We Access

Calendar Work Manager uses Google OAuth 2.0 to access Google Calendar data only after the user explicitly grants permission.

Permission (Scope) Description Purpose
https://www.googleapis.com/auth/calendar Full access to Google Calendar Read calendar list and calendar metadata
https://www.googleapis.com/auth/calendar.events Access to calendar events Create, view, update, and delete calendar events

Types of data accessed:

  • Calendar data: calendar ID, calendar name
  • Event data: event title, date/time, description, location

2. How We Access Data

All access to Google Calendar data occurs through Google's official APIs using OAuth 2.0 authentication. Users must explicitly approve access before any data is accessed.

3. Purpose of Data Use

The accessed data is used strictly to provide the core functionality of the extension:

  • Creating work tasks as Google Calendar events
  • Displaying available calendars for user selection
  • Preventing duplicate or conflicting events
  • Improving task scheduling convenience

The extension does NOT use data for:

  • Advertising or marketing
  • User tracking or analytics
  • Selling or sharing data with third parties

4. Data Storage and Retention

  • Calendar Work Manager does not operate any external servers.
  • No personal data is stored outside the user's browser.
  • OAuth access tokens are stored locally in the browser and are removed when the extension is uninstalled.

5. Data Sharing

Calendar Work Manager does not share user data with any third parties.

Data is transmitted only between the user's browser and Google servers as required to interact with Google Calendar APIs.

6. User Control and Rights

Users may:

  • Revoke access at any time via Google Account Permissions
  • Remove all locally stored data by uninstalling the extension

7. Google API Services Compliance

Calendar Work Manager complies with the Google API Services User Data Policy, including the Limited Use requirements.

Data obtained through Google APIs is used solely to provide the app’s user-facing features and is not transferred, sold, or used for any other purpose.
